  rightIcon?: ReactNode;

# 8. Accessibility Requirements

## 8.1 WCAG 2.1 AA Compliance

- Color contrast ratio minimum 4.5:1 for text
- All interactive elements keyboard accessible
- Focus indicators visible
- Screen reader compatible
- Reduced motion option

## 8.2 Keyboard Shortcuts

| Action | Shortcut |
|--------|----------|
| End turn | Space |
| Attack | A |
| Cast spell | S |
| Move mode | M |
| Open inventory | I |
| Open character | C |
| Toggle grid | G |
| Zoom in | + |
| Zoom out | - |
| Escape/Cancel | Esc |

## 8.3 Colorblind Mode

- Condition icons use shapes + colors
- Damage types use patterns
- Team indicators use shapes (circle = ally, diamond = enemy)
- Health uses both color and percentage text
