# Document 34: Video & Cutscene Storyboard

## Purpose
This document provides **storyboards for all video sequences, cutscenes, and cinematic moments** in the D&D Digital Board Game. Use these specifications when implementing video playback, camera animations, and narrative sequences.

---

## Video System Overview

### Video Categories

| Category | Duration | Format | Trigger |
|----------|----------|--------|---------|
| App Intro | 5-8s | WebM/MP4 | App launch |
| Campaign Intro | 15-30s | WebM/MP4 | Campaign start |
| Chapter Intro | 10-20s | WebM/MP4 | Chapter transition |
| Boss Intro | 8-15s | WebM/MP4 | Boss encounter start |
| Victory Sequence | 10-15s | WebM/MP4 | Combat victory |
| Defeat Sequence | 8-12s | WebM/MP4 | TPK |
| Level Up | 5-8s | CSS/WebM | Character level up |
| Critical Hit | 2-4s | CSS/Canvas | Natural 20 attack |
| Death Save | 3-5s | CSS/Canvas | Death save roll |
| Cutscene Dialog | Variable | Parallax HTML | Story beats |

```typescript
interface VideoConfig {
  format: 'webm' | 'mp4';

    desktop: '4Mbps',

}
```

interface DialogScene {
  id: string;
  background: ParallaxBackground;
  characters: DialogCharacter[];
  dialog: DialogNode[];
}

interface ParallaxBackground {
  layers: {
    src: string;
    z: number;
    parallaxFactor: number;
    animation?: {
      type: 'drift' | 'float' | 'sway';
      amplitude: number;
      frequency: number;
    };
  }[];
  ambientParticles?: {
    type: 'dust' | 'fireflies' | 'snow' | 'embers';
    count: number;
    color: string;
  };
}

interface DialogCharacter {
  id: string;
  name: string;
  portrait: string;

    neutral: string;

    entry: 'slide' | 'fade' | 'pop';

}

  id: string;
  speaker: string;

    text: string;
    next: string;
    condition?: string;
  }[];
  next?: string;
}
